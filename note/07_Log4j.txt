# Log4j

 - 자바의 로깅 라이브러리
 - sysout보다 체계적인 방식으로 로그를 남길 수 있다
 - 각 로그마다 등급을 설정하고 일정 등급 이상의 로그만 볼 수 있도록 설정한다
 - Logger : 해당 클래스에서 발생하는 로그들을 수집하여 Appender로 전달하는 객체
 - Appender : 실제로 로그를 기록하는 객체(파일, 콘솔, DB 등...)
 - Layout : 로그를 어떤 모양으로 기록할지 설정하는 객체

# Logging Level

 - OFF		: 모든 등급의 로그를 끔
 - FATAL	: 아주 심각한 오류만 보여줌
 - ERROR	: 심각한 오류 이상의 것만 보여줌
 - WARN		: 경고 이상의 오류만 보여줌
 - INFO		: 정보 이상의 오류만 보여줌
 - DEBUG	: 디버깅 단계에서 보여질 로그
 - TRACE	: 버그 추적 단계에서 보여질 상세 로그
 - ALL		: 모든 로그를 켬
